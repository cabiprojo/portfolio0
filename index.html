Craigslist Redesigned

In this front-end development focused project, I drew inspiration from the outdated Craigslist website and transformed it into a modern and visually appealing site. The revamp involved utilizing Google API, implementing scroll animations, and incorporating modal popups. The project ensures a fully responsive site and all links and buttons excluding those to create or browse postings are functional.

Piano Recorder

This was a simple, fun project focused on the backend where users can press keys on the keyboard that correspond to piano keys. It provides an option to record and play the keys, and when the playback is initiated, the piano plays the recorded keys automatically. The project utilizes Node.js, a backend environment, to store recordings in a database, along with audio files and animations.
